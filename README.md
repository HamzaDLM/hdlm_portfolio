## Development Setup

### Clone the github repo
```
git clone [url]
```
### Install dependency
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```
